# flutter-tutorials
Code written during Flutter's tutorial (https://docs.flutter.dev/learn/pathway/tutorial)

## Birdle
Birdle is a Flutter-based word-guessing game featuring:

- Intuitive UI with Flutter widgets
- On-screen keyboard and input validation
- Real-time feedback with color-coded hints
- Smooth letter and result animations
- State persistence and responsive design
- Light/dark mode support

## Wikipedia reader
Wikipedia reader is a Flutter-based app that fetches a random article.

- HTTP requests
- MVVM architecture
- ChangeNotifier to manage state and update UI
- Responsive user interfaces
